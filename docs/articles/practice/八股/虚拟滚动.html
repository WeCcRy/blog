不定高虚拟列表

这个页面演示的是“复用固定 DOM 池 + 替换内容 + 不定高测量”的虚拟列表实现。 你可以直接滚动左侧列表，观察节点不会无限增加，但内容会不断复用和重排。

总数据量 0
复用节点数 0
可视区高度 0

滚动区域

只渲染当前可见项，滚动时替换内容

实现要点

1. `pool` 是固定数量的 DOM 节点，不会随着数据量增长。

2. `translateY` 负责把节点移动到它应该出现的位置。

3. 不定高 item 通过 `getBoundingClientRect()` 测量后回写前缀和。

4. 这里用的是“替换节点内容”，不是向列表里持续 `push` 新节点。

scrollTop -> 二分查找 -> startIndex -> 复用 DOM -> 替换内容 -> 测量高度

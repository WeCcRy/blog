иҮӘе®ҡд№үе…ғзҙ е…ұдә«

echartsиҒ”еҠЁе…¶д»–з»„д»¶

On this page
  1. жӢүеҸ–и§ҰзӮ№й…ҚзҪ®пјҲи§„еҲҷеұӮпјү зі»з»ҹеҗҜеҠЁеҗҺе…ҲеҒҡзҡ„жҳҜжӢҝвҖңи§ҰеҸ‘и§„еҲҷвҖқпјҢдёҚжҳҜжӢҝеұ•зӨәеҶ…е®№гҖӮи§„еҲҷжқҘжәҗжңүдёӨжқЎи·Ҝеҫ„пјҡ зј“еӯҳи·Ҝеҫ„пјҡеҰӮжһңй…ҚзҪ®дәҶжң¬ең°еӯҳеӮЁ keyпјҢдјҳе…ҲиҜ»жң¬ең°зј“еӯҳ иҜ»еҸ–еҗҺж ЎйӘҢзүҲжң¬еҸ·е’Ңз»“жһ„еҗҲжі•жҖ§ ж ЎйӘҢйҖҡиҝҮзӣҙжҺҘиҝӣе…ҘеҗҺз»ӯжөҒзЁӢ е®һж—¶и·Ҝеҫ„пјҡзј“еӯҳдёҚеӯҳеңЁ/дёҚеҗҲжі•ж—¶пјҢи°ғз”Ёй…ҚзҪ®жҺҘеҸЈе®һж—¶иҺ·еҸ– й…ҚзҪ®иҝ”еӣһзҡ„ж ёеҝғдҝЎжҒҜпјҡ и§ҰзӮ№еҲ—иЎЁпјҲжҜҸдёӘи§ҰзӮ№жңүйЎөйқўж ҮиҜҶгҖҒи§ҰзӮ№ж ҮиҜҶгҖҒжқЎд»¶еҲ—иЎЁпјү зӯ–з•ҘпјҲз»ҲжӯўеһӢ or иҰҶзӣ–еһӢпјү иҝҷйҮҢзҡ„вҖңи§ҰзӮ№вҖқжң¬иҙЁжҳҜдёҖдёӘи§„еҲҷеҚ•е…ғпјҡеҪ“е®ғиҮӘе·ұзҡ„жқЎд»¶еҲ—иЎЁе…ЁйғЁж»Ўи¶іж—¶пјҢжүҚжңүиө„ж јиҝӣе…ҘеҶ…е®№иҜ·жұӮйҳ¶ж®өгҖӮ
  2. иҝҮж»ӨйЎөйқўи§ҰзӮ№пјҲдҪңз”Ёеҹҹж”¶ж•ӣпјү жӢҝеҲ°е…ЁйҮҸй…ҚзҪ®еҗҺпјҢдёҚдјҡе…Ёи·‘гҖӮе…ҲжҢүеҪ“еүҚйЎөйқўж ҮиҜҶиҝҮж»Өпјҡ еҸӘдҝқз•ҷвҖңеұһдәҺеҪ“еүҚйЎөйқўвҖқзҡ„и§ҰзӮ№ еӯҳе…ҘвҖңеҫ…жЈҖжөӢи§ҰзӮ№жұ вҖқ иҝҷдёҖжӯҘзӣ®зҡ„еҫҲжҳҺзЎ®пјҡ йҷҚдҪҺеҲӨе®ҡејҖй”Җ йҒҝе…Қи·ЁйЎөйқўи§ҰзӮ№иҜҜи§ҰеҸ‘ и®©йЎөйқўеҶ…еҲӨе®ҡйҖ»иҫ‘еҸӘеӣҙз»•жң¬йЎөи§„еҲҷиҝҗиЎҢ
  3. еҲқе§ӢеҢ–и§ҰеҸ‘еҷЁпјҲдәӢд»¶жәҗжҢӮиҪҪпјү иҝҮж»ӨеҗҺпјҢзі»з»ҹдјҡжҠҠи§ҰзӮ№жқЎд»¶жӢҶжҲҗеҸҜзӣ‘еҗ¬зҡ„и§ҰеҸ‘жәҗпјҢеҪ“еүҚдё»иҰҒдёӨзұ»пјҡ 3.1 еҒңз•ҷж—¶й•ҝжқЎд»¶пјҲtimeoutпјү жң¬ең°з»ҙжҠӨдёҖдёӘе®ҡж—¶еҷЁпјҢе®ҡж—¶жӣҙж–°еҪ“еүҚеҒңз•ҷж—¶й•ҝзҠ¶жҖҒ жҜҸж¬Ўжӣҙж–°ж—¶и§ҰеҸ‘з»ҹдёҖеҲӨе®ҡе…ҘеҸЈ 3.2 ж»ҡеҠЁеҒңжӯўжқЎд»¶пјҲscroll_stopпјү йЎөйқўж»ҡеҠЁз»“жқҹж—¶и§ҰеҸ‘ еҜ№жғҜжҖ§ж»‘еҠЁеҒҡеҢәеҲҶпјҡиӢҘд»ҚжңүжғҜжҖ§йҖҹеәҰпјҢе»¶иҝҹдёҖе°Ҹж®өж—¶й—ҙеҶҚи§ҰеҸ‘(appз«ҜеҸҜд»ҘеҲӨж–ӯжҳҜеҗҰжңүж»ҡеҠЁжғҜжҖ§) и§ҰеҸ‘еҗҺиҝӣе…Ҙз»ҹдёҖеҲӨе®ҡе…ҘеҸЈпјҢж Үи®°ж»ҡеҠЁеҒңжӯўе·ІиҫҫжҲҗ е…ій”®зӮ№пјҡtimeout е’Ң scroll_stop еҸӘжҳҜвҖңи§ҰеҸ‘жәҗвҖқпјҢзңҹжӯЈеҲӨе®ҡйғҪиө°еҗҢдёҖдёӘжқЎд»¶еј•ж“Һе…ҘеҸЈгҖӮ
  4. ж»Ўи¶ізҠ¶жҖҒиЎЁпјҲиҝҗиЎҢж—¶зҠ¶жҖҒпјү зі»з»ҹз»ҙжҠӨдёҖд»ҪвҖңе·Іж»Ўи¶ізҠ¶жҖҒиЎЁвҖқпјҢз”ЁдәҺи·ЁдәӢд»¶зҙҜз§ҜжқЎд»¶гҖӮжҜ”еҰӮпјҡ scroll_stop жҳҜеҗҰе·ІеҸ‘з”ҹ еҪ“еүҚе·ІиҫҫеҲ°зҡ„ timeout йҳҲеҖјжҳҜеӨҡе°‘ жҜҸж¬ЎдәӢд»¶еҸ‘з”ҹпјҢеҸӘжҳҜвҖңжӣҙж–°зҠ¶жҖҒ + и§ҰеҸ‘йҮҚеҲӨвҖқпјҢиҖҢдёҚжҳҜдёҖж¬ЎжҖ§жӢҚи„‘иўӢеҶіе®ҡеұ•зӨәгҖӮ иҝҷдҪҝеҫ—жқЎд»¶йЎәеәҸж— е…іпјҡ е…Ҳж»ҡеҠЁеҗҺеҲ°ж—¶й•ҝпјҢиғҪи§ҰеҸ‘ е…ҲеҲ°ж—¶й•ҝеҗҺж»ҡеҠЁпјҢд№ҹиғҪи§ҰеҸ‘
  5. жқЎд»¶еҲӨе®ҡпјҲжҜҸдёӘи§ҰзӮ№еҒҡ AND/everyпјү з»ҹдёҖеҲӨе®ҡе…ҘеҸЈжҜҸж¬Ўжү§иЎҢж—¶йғҪдјҡпјҡ жӣҙж–°ж»Ўи¶ізҠ¶жҖҒиЎЁпјҲеҶҷе…Ҙжң¬ж¬ЎдәӢд»¶зҠ¶жҖҒпјү ж №жҚ®зӯ–з•Ҙе…ҲеҒҡзҹӯи·ҜеҲӨж–ӯпјҲеҰӮз»ҲжӯўеһӢдё”е·Іеұ•зӨәиҝҮеҲҷзӣҙжҺҘиҝ”еӣһпјү йҒҚеҺҶеҫ…жЈҖжөӢи§ҰзӮ№жұ  еҜ№жҜҸдёӘи§ҰзӮ№зҡ„жқЎд»¶еҲ—иЎЁеҒҡвҖңе…ЁйғЁж»Ўи¶івҖқеҲӨж–ӯпјҲevery/ANDпјү ж»ҡеҠЁжқЎд»¶пјҡзңӢж Үи®°жҳҜеҗҰеӯҳеңЁ ж—¶й•ҝжқЎд»¶пјҡзңӢеҪ“еүҚиҫҫжҲҗеҖјжҳҜеҗҰ >= й…ҚзҪ®йҳҲеҖј пјҲи®°еҪ•е·Із»ҸеңЁеҪ“еүҚйЎөйқўеҒңз•ҷеӨҡд№…пјҢжң¬ең°з»ҙжҠӨдёҖдёӘstateпјҢз„¶еҗҺе®ҡж—¶жӣҙж–°пјҢжҜҸж¬Ўжӣҙж–°зҡ„ж—¶еҖҷcheckConditionsпјү ж”¶йӣҶе‘Ҫдёӯи§ҰзӮ№еҲ—иЎЁ еҸӘжңүвҖңеҚ•и§ҰзӮ№еҶ…жүҖжңүжқЎд»¶йғҪж»Ўи¶івҖқжүҚз®—е‘ҪдёӯпјҢдёҚжҳҜд»»дёҖжқЎд»¶ж»Ўи¶ігҖӮ
  6. е‘Ҫдёӯи§ҰзӮ№ж¶Ҳиҙ№пјҲйҳІйҮҚеӨҚзҡ„з¬¬дёҖеұӮпјү еҪ“жҹҗж¬ЎеҲӨе®ҡеҫ—еҲ°е‘ҪдёӯеҲ—иЎЁеҗҺпјҡ иҝҷдәӣи§ҰзӮ№дјҡз«ӢеҚід»Һеҫ…жЈҖжөӢи§ҰзӮ№жұ з§»йҷӨ з„¶еҗҺеҶҚеҸ‘иө·еҶ…е®№иҜ·жұӮ вҖңе…Ҳз§»йҷӨеҗҺиҜ·жұӮвҖқеҫҲйҮҚиҰҒпјҡ йҳІжӯўеҗҢдёҖи§ҰзӮ№иў«еҗҺз»ӯдәӢд»¶йҮҚеӨҚе‘Ҫдёӯ йҷҚдҪҺйҮҚеӨҚиҜ·жұӮжҰӮзҺҮ
  7. иҜ·жұӮи§ҰзӮ№е®һйҷ…ж•°жҚ®пјҲеҶ…е®№еұӮпјү е‘ҪдёӯеҗҺжүҚиҜ·жұӮеұ•зӨәеҶ…е®№жҺҘеҸЈпјҢе…ҘеҸӮжҳҜе‘Ҫдёӯи§ҰзӮ№ж ҮиҜҶйӣҶеҗҲгҖӮеҗҺз«ҜеңЁиҝҷдёӘйҳ¶ж®өеҒҡзңҹжӯЈвҖңйҮҚйҖ»иҫ‘вҖқпјҡ йў‘жҺ§еҲӨж–ӯпјҲжҳҜеҗҰе…Ғи®ёеұ•зӨәпјү дёҡеҠЎж•°жҚ®еҲӨж–ӯпјҲиҜҘи§ҰзӮ№еҪ“еүҚжҳҜеҗҰжңүж•Ҳпјү иҝ”еӣһеҸҜеұ•зӨәеҶ…е®№пјҲж–ҮжЎҲгҖҒж ·ејҸгҖҒж–№еҗ‘гҖҒж—¶й•ҝгҖҒи·іиҪ¬зӯүпјү еүҚз«ҜжӢҝеҲ°з»“жһңеҗҺйҖүжӢ©еұ•зӨәеҜ№иұЎе№¶жёІжҹ“гҖӮ
  8. еұ•зӨәзӯ–з•ҘдёҺе№¶еҸ‘йҳІйҮҚпјҲдёӨеұӮдҝқйҷ©пјү зі»з»ҹж”ҜжҢҒдёӨз§Қзӯ–з•Ҙпјҡ з»ҲжӯўеһӢпјҡйЎөйқўеұ•зӨәдёҖж¬ЎеҗҺпјҢдёҚеҶҚеӨ„зҗҶеҗҺз»ӯи§ҰзӮ№ иҰҶзӣ–еһӢпјҡеҗҺз»ӯе‘ҪдёӯеҸҜз»§з»ӯеұ•зӨәпјҲе…Ғи®ёиҰҶзӣ–пјү еңЁз»ҲжӯўеһӢдёӢпјҢеҒҡдәҶеҸҢеұӮйҳІз«һжҖҒпјҡ иҜ·жұӮеүҚжӢҰжҲӘпјҡиӢҘе·Іеұ•зӨәпјҢеҗҺз»ӯе‘ҪдёӯдёҚеҶҚеҸ‘иҜ·жұӮ е“Қеә”еҗҺжӢҰжҲӘпјҡеҚідҪҝе№¶еҸ‘иҜ·жұӮе·ІеҸ‘еҮәпјҢеҗҺиҝ”еӣһиҖ…д№ҹдјҡиў«дёўејғ иҝҷж ·еҸҜд»ҘйҒҝе…ҚвҖңеҮ д№ҺеҗҢж—¶е‘ҪдёӯеҜјиҮҙеӨҡж¬Ўеұ•зӨәвҖқзҡ„й—®йўҳгҖӮ
  9. йЎөйқўдәӢд»¶иҒ”еҠЁпјҲж»ҡеҠЁ/зҰ»ејҖпјү и§ҰзӮ№зі»з»ҹжң¬иә«е№¶дёҚзӣҙжҺҘжӢҘжңүйЎөйқўеҺҹз”ҹдәӢд»¶пјҢеӣ жӯӨйҖҡиҝҮжҺҘе…ҘеұӮжЎҘжҺҘпјҡ йЎөйқўж»ҡеҠЁз»“жқҹ -> йҖҡзҹҘи§ҰзӮ№зі»з»ҹиҝӣиЎҢ scroll_stop еҲӨе®ҡ йЎөйқўзҰ»ејҖ -> дё»еҠЁе…ій—ӯеҪ“еүҚеұ•зӨә е№¶дё”жЎҘжҺҘж–№ејҸжҳҜвҖңе…Ҳжү§иЎҢдёҡеҠЎеҺҹйҖ»иҫ‘пјҢеҶҚжү§иЎҢи§ҰзӮ№иЎҘе……йҖ»иҫ‘вҖқпјҢе°ҪйҮҸдёҚз ҙеқҸеҺҹйЎөйқўиЎҢдёәгҖӮ
  10. з”ҹе‘Ҫе‘Ёжңҹжё…зҗҶпјҲйҳІи„ҸзҠ¶жҖҒпјү йЎөйқўеҚёиҪҪж—¶дјҡз»ҹдёҖжё…зҗҶпјҡ еҫ…жЈҖжөӢи§ҰзӮ№жұ  ж»Ўи¶ізҠ¶жҖҒиЎЁ жүҖжңү timeout е®ҡж—¶еҷЁ е·Іеұ•зӨәж Үи®°гҖҒеҪ“еүҚеұ•зӨәж ҮиҜҶгҖҒиҮӘеҠЁе…ій—ӯе®ҡж—¶еҷЁ зЎ®дҝқдёӢж¬Ўиҝӣе…ҘйЎөйқўд»Һе№ІеҮҖзҠ¶жҖҒйҮҚж–°ејҖе§ӢпјҢйҒҝе…Қи·Ёж¬ЎжұЎжҹ“гҖӮ
  11. е®һзҺ°ж–№жі• зі»з»ҹжҸҗдҫӣдәҶдёҖдёӘж ёеҝғе®№еҷЁ TouchPointProviderгҖӮеңЁеҺҹжңүйЎөйқўеӨ–еұӮиҝӣиЎҢеҢ…иЈ№пјҢеҚіеҸҜйӣ¶дҫөе…Ҙең°жіЁе…Ҙи§ҰзӮ№йҖ»иҫ‘е’Ңеј№зӘ—иғҪеҠӣгҖӮ

11.1 еҶ…йғЁе®һзҺ°еҺҹзҗҶ вҖӢ

TouchPointProvider жң¬иҙЁдёҠжҳҜдёҖдёӘй«ҳйҳ¶з»„д»¶е®№еҷЁпјҢе®ғйҖҡиҝҮжёІжҹ“ props.children жқҘдҝқжҢҒеҺҹжңүдёҡеҠЎйЎөйқўзҡ„жёІжҹ“пјҢе№¶еҗҢжӯҘжҢӮиҪҪдёҖдёӘзӢ¬з«Ӣзҡ„жёІжҹ“еј•ж“ҺиҠӮзӮ№гҖӮ

jsx
// TouchPointProvider дјӘд»Јз Ғ
const TouchPointProvider = ({ children, pageId }) => {
  return (
    <TouchContext.Provider value={{ activeData }}>
      {/* жёІжҹ“еҺҹдёҡеҠЎйЎөйқўпјҡдҝқжҢҒе…¶еҺҹжңүеұӮзә§е’Ңз”ҹе‘Ҫе‘Ёжңҹ */}
      {children}

      {/* и§ҰзӮ№еұ•зӨәеұӮпјҡдёҺдёҡеҠЎйЎөйқўеңЁ DOM дёҠеұһдәҺеҗҢзә§жҲ–йҖҡиҝҮ Portal жҢӮиҪҪ */}
      <TouchPointModal />
    </TouchContext.Provider>
  );
};
